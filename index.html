Ringkasan Materi

Materi ini dirancang untuk latihan mengalir dari nol sampai jadi 1 website (1 halaman) memakai HTML + CSS. Setiap tahap punya:

  • Tujuan tahap (apa yang dipelajari)

üìå Kompetensi yang dilatih

  • Struktur HTML: head, body, meta, title
  • Anchor internal: href="#id"
  • Link eksternal: target="_blank"
  • Gambar: img + alt
  • Tabel: table, tr, th, td
  • Form: form, input, textarea, required
  • CSS: selector, box model, flexbox, hover

üß© Hasil akhir halaman

  1. Header + Navbar rapi (Flexbox)
  2. Section Tentang + link referensi
  3. Section Galeri (Flexbox) + hover
  4. Section Data (Tabel rapi)
  5. Section Kontak (Form rapi)
  6. Footer sederhana

Tahap 1

Tahap 1 ‚Äî Kerangka HTML dan Judul Dasar

Apa yang dipelajari?

  • Struktur HTML minimal: <head>, <body>
  • Metadata dasar: meta charset, viewport
  • Judul halaman: title
  • Konten awal: h1 dan p

Praktik siswa

  1. Buat file index.html.
  2. Copy kode di bawah, simpan.
  3. Buka di browser dan pastikan judul + paragraf tampil.

Konsep utama

  • Struktur dokumen HTML harus lengkap agar browser memahami halaman dengan benar.
  • Heading digunakan untuk judul, sedangkan paragraf untuk isi teks.

Tag yang dipelajari pada tahap ini

Tag/AtributKegunaanCatatan Penulisan
<!DOCTYPE html>Menandai dokumen sebagai HTML5.Ditulis paling atas, 1 kali.
<html lang="id">Elemen root (paling luar) + bahasa dokumen.Gunakan lang sesuai bahasa halaman.
<head>Berisi informasi halaman (metadata).Tidak tampil di halaman.
<meta charset="UTF-8">Mengatur encoding agar teks terbaca benar.Wajib untuk karakter Indonesia.
<meta name="viewport" content="width=device-width, initial-scale=1.0">Membuat tampilan responsif di HP.Umum dipakai di web modern.
<title>Judul tab browser.Singkat dan jelas.
<body>Semua konten yang terlihat pengguna.Isi halaman ditulis di sini.
<h1>Judul utama halaman.Biasanya 1 kali per halaman.
<p>Paragraf teks.Dipakai untuk kalimat/penjelasan.

Contoh penulisan

Kerangka HTML
<!DOCTYPE html>
<html lang="id">
  <head>...</head>
  <body>...</body>
</html>
Urutan: DOCTYPE ‚Üí html ‚Üí head ‚Üí body.
Heading & paragraf
<h1>Judul Utama</h1>
<p>Ini paragraf.</p>
Gunakan <h1> untuk judul utama dan <p> untuk isi.

Legenda: Bagian baru pada tahap ini

Tahap 2 ‚Äî Menu, Tautan Internal (Anchor), dan ID Bagian

Apa yang dipelajari?

  • Membuat menu sederhana menggunakan tag <a>
  • Link internal ke bagian halaman: href="#tentang"
  • Memberi identitas section dengan id

Praktik siswa

  1. Replace isi index.html dengan kode Tahap 2.
  2. Klik menu ‚ÄúTentang/Galeri/Data/Kontak‚Äù ‚Üí harus loncat ke bagian terkait.
  3. Ubah isi placeholder sesuai kebutuhan.

Konsep utama

  • Membuat menu navigasi menggunakan tautan (<a>) yang mengarah ke bagian tertentu dalam halaman.
  • Memakai id pada judul bagian agar bisa dituju oleh tautan #id.

Tag yang dipelajari pada tahap ini

Tag/AtributKegunaanCatatan Penulisan

id="tentang"Penanda unik pada elemen.Tidak boleh sama dalam 1 halaman.
<b>Teks tebal (bold).Untuk penekanan sederhana.
<i>Teks miring (italic).Untuk istilah/penekanan.

Cara penulisan yang harus dipahami

Tautan internal (anchor)

Klik menu akan lompat ke elemen yang memiliki id="galeri".
Target id pada section

Nilai id harus sama persis dengan target di href.

Tahap 5 ‚Äî Tabel (table, tr, th, td)

Apa yang dipelajari?

  • Membuat tabel data dengan table, tr, th, td
  • Membedakan header kolom (th) dan isi data (td)

Praktik siswa

  1. Replace file dengan kode Tahap 5.
  2. Tambah 1 baris data lagi (jadi 5 baris).

Konsep utama

  • Menyajikan data rapi menggunakan tabel.
  • Memahami peran <table>, <tr>, <th>, dan <td>.

Tag yang dipelajari pada tahap ini

Tag/AtributKegunaanCatatan Penulisan
<table>Wadah/kerangka tabel.Berisi baris-baris tabel.
<tr>Table Row = satu baris.Di dalamnya ada sel (th atau td).
<th>Table Header = judul kolom.Dipakai pada baris judul kolom.
<td>Table Data = isi sel.Dipakai pada baris data.
border / cellpaddingGaris & jarak dalam tabel.Atribut lama; untuk latihan diperbolehkan. Nanti bisa dirapikan dengan CSS.

Cara penulisan tiap tag (harus dipahami)

1) Kerangka tabel
<table>
  ...baris tabel...
</table>
Semua baris tabel berada di dalam <table>.
2) Membuat 1 baris (TR)
<tr>
  <td>Kolom 1</td>
  <td>Kolom 2</td>
</tr>
<tr> berisi beberapa sel. Jumlah sel = jumlah kolom.
3) Baris judul kolom (TH)
<tr>
  <th>No</th>
  <th>Kegiatan</th>
</tr>
Gunakan <th> untuk judul kolom di baris pertama.
4) Baris data (TD)
<tr>
  <td>1</td>
  <td>Latihan</td>
</tr>
Gunakan <td> untuk data/isi tabel.

Tahap 6 ‚Äî Formulir (Form) dan Komponen Input

Apa yang dipelajari?

  • Struktur form: <form action method>
  • Input dasar: text, email, radio, checkbox
  • Validasi sederhana: required, type="email"
  • Input teks panjang: textarea

Praktik siswa

  1. Replace file dengan kode Tahap 6.
  2. Coba kirim form dengan email tidak valid ‚Üí harus ditolak browser.
  3. Tambah 1 input lagi (misal: select ‚ÄúKelas‚Äù).

Konsep utama

  • Membuat formulir untuk input data pengguna.
  • Mengenal tipe input: teks, email, radio, checkbox, textarea, dan tombol kirim.

Tag yang dipelajari pada tahap ini

Tag/AtributKegunaanCatatan Penulisan
<form>Wadah formulir input.Memiliki action dan method.
action="..."Tujuan pengiriman data form.Untuk latihan boleh #.
method="GET/POST"Cara pengiriman data.GET tampil di URL; POST untuk data sensitif.
<input>Elemen input.Tentukan jenis input dengan type.
type="text"Input teks biasa.Untuk nama, kelas, dll.
type="email"Input email.Browser membantu validasi format email.
type="radio"Pilihan satu dari beberapa opsi.Gunakan name sama untuk satu grup.
type="checkbox"Pilihan lebih dari satu.Boleh memilih beberapa opsi.
<textarea>Input teks panjang.Untuk pesan/keterangan.
requiredWajib diisi.Jika kosong, form tidak bisa dikirim.
name="..."Nama variabel data.Dipakai sebagai kunci saat data dikirim.

Contoh penulisan penting

</form>

placeholder memberi contoh, required mewajibkan.

<input type="radio" name="kategori" value="guru"> Guru

Checkbox dapat dipilih lebih dari satu.
Textarea
<textarea name="pesan" rows="4"></textarea>
Untuk teks panjang. rows menentukan tinggi awal.
Tombol kirim
<button type="submit">Kirim</button>
submit mengirim form sesuai action/method.

Tahap 8

Tahap 8 ‚Äî Flexbox (Navbar dan Galeri) serta Efek Hover

Apa yang dipelajari?

  • Navbar rapi dengan Flexbox: display:flex, justify-content, align-items, gap
  • Galeri rapi dengan Flexbox + flex-wrap
  • Efek interaksi (hover): :hover, transition, transform, opacity
  • Merapikan tabel & form dengan CSS

Praktik siswa

  1. Replace file dengan kode Tahap 8.
  2. Pastikan navbar rapi dan galeri responsif (jika layar sempit, gambar turun baris).
  3. Coba hover pada menu dan gambar.

Konsep utama

  • Merapikan layout dengan Flexbox untuk navbar dan galeri.
  • Menambahkan efek interaksi (hover dan transition) agar tampilan lebih modern.

Tag yang dipelajari pada tahap ini

Tag/AtributKegunaanCatatan Penulisan
display: flexMengaktifkan flexbox.Elemen anak dapat disusun sejajar dan responsif.
justify-contentMengatur posisi horizontal.Contoh: space-between.
align-itemsMengatur posisi vertikal.Contoh: center.
gapJarak antar item flex.Lebih rapi daripada memberi margin manual.
flex-wrapMembolehkan item turun baris.Penting untuk layar kecil.
:hoverEfek saat kursor di atas elemen.Untuk menu/gambar/tombol.
transitionAnimasi perpindahan halus.Contoh: 0.2s agar tidak kaku.

Contoh penulisan yang harus dipahami

Flexbox navbar
.navbar { display: flex; justify-content: space-between; align-items: center; }
Logo di kiri dan menu di kanan.
Flexbox galeri
.galeri { display: flex; gap: 12px; flex-wrap: wrap; }
Gambar rapi dan turun baris saat layar sempit.
Hover + transition
.galeri img { transition: 0.2s; }
.galeri img:hover { transform: scale(1.03); }
Efek halus saat gambar disentuh kursor.
