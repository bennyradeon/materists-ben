Ringkasan Materi

Materi ini dirancang untuk latihan mengalir dari nol sampai jadi 1 website (1 halaman) memakai HTML + CSS. Setiap tahap punya:

  • Tujuan tahap (apa yang dipelajari)
  • Praktik (apa yang dilakukan siswa)
  • Kode lengkap (index.html) yang sudah berisi tahap sebelumnya + tambahan tahap sekarang
Target akhir pembelajaran: 1 file index.html berisi Navbar (Flexbox), Tentang, Galeri, Tabel, Form, dan Footer dengan CSS rapi.

üìå Kompetensi yang dilatih

  • Struktur HTML: head, body, meta, title
  • Anchor internal: href="#id"
  • Link eksternal: target="_blank"
  • Gambar: img + alt
  • Tabel: table, tr, th, td
  • Form: form, input, textarea, required
  • CSS: selector, box model, flexbox, hover

üß© Hasil akhir halaman

  1. Header + Navbar rapi (Flexbox)
  2. Section Tentang + link referensi
  3. Section Galeri (Flexbox) + hover
  4. Section Data (Tabel rapi)
  5. Section Kontak (Form rapi)
  6. Footer sederhana

Petunjuk Penggunaan di Kelas

  1. Guru menampilkan materi ini (website dokumentasi).
  2. Siswa membuat folder latihan dan file index.html.
  3. Siswa copy-paste kode tahap 1, simpan, buka di browser.
  4. Setelah paham, siswa replace isi file dengan kode tahap 2, simpan, refresh.
  5. Ulangi sampai tahap 8 ‚Üí website final jadi.
Perhatian
Penting: Karena materi ini bersifat berurutan, peserta didik tidak perlu menambahkan file lain terlebih dahulu. Semua latihan dilakukan di index.html sampai tahap akhir.

Tahap 1

Tahap 1 ‚Äî Kerangka HTML dan Judul Dasar

Apa yang dipelajari?

  • Struktur HTML minimal: <head>, <body>
  • Metadata dasar: meta charset, viewport
  • Judul halaman: title
  • Konten awal: h1 dan p

Praktik siswa

  1. Buat file index.html.
  2. Copy kode di bawah, simpan.
  3. Buka di browser dan pastikan judul + paragraf tampil.

Kode lengkap (Tahap 1)

  <p>Website ini dibuat untuk latihan HTML dan CSS.</p>

</body>
</html>

Tahap 8 ‚Äî Flexbox (Navbar dan Galeri) serta Efek Hover

Apa yang dipelajari?

  • Navbar rapi dengan Flexbox: display:flex, justify-content, align-items, gap
  • Galeri rapi dengan Flexbox + flex-wrap
  • Efek interaksi (hover): :hover, transition, transform, opacity
  • Merapikan tabel & form dengan CSS

Praktik siswa

  1. Replace file dengan kode Tahap 8.
  2. Pastikan navbar rapi dan galeri responsif (jika layar sempit, gambar turun baris).
  3. Coba hover pada menu dan gambar.

Template Akhir (Untuk Ujian)

Ini adalah versi final yang sama konsepnya dengan Tahap 8, namun ditulis ulang agar lebih rapi dan siap dipakai sebagai template ujian. Kamu bisa pakai ini sebagai starter file untuk siswa.

Informasi
Rekomendasi pelaksanaan ujian: Guru dapat meminta peserta didik menyesuaikan isi berikut: judul, deskripsi, minimal 5 baris tabel, minimal 5 gambar, dan menambah 1 input baru pada form.
Lihat Tahap 8 (kode final berantai) Lompat ke Checklist

Checklist Capaian Pembelajaran

HTML

  • Membuat struktur HTML lengkap
  • Membuat menu anchor #id
  • Membuat link eksternal tab baru
  • Menampilkan gambar + alt
  • Membuat tabel minimal 4 baris
  • Membuat form dengan email required

CSS

  • Menggunakan selector element, class, dan id
  • Menggunakan box model (margin/padding/border)
  • Membuat Flexbox navbar atau galeri
  • Membuat efek hover pada menu/gambar
  • Merapikan tabel dan form
Catatan
Latihan penguatan: Mintalah peserta didik membuat tema berbeda (Profil Kelas / Ekskul / Event) dengan tetap memakai struktur final yang sama.

Penutup

Materi ini dapat ditampilkan secara lokal (membuka index.html) atau diunggah ke hosting sekolah. Apabila diperlukan, materi ini dapat dikembangkan dengan fitur tambahan seperti tombol salin kode (Copy Code) pada setiap tahap menggunakan JavaScript sederhana.
